# Plum
Plum is a simple compiled stack-based programming language

> Note: Plum needs gcc on the system to assemble and link the generated assembly code.

The full language only compiles to AArch64 ARM right now. See the code examples and compile them to better understand the language syntax and behaviour.

## Build a .plum file

```
python plum.py <.plum file>
```

Compiling a file will create a .s file on the system.
